## 🚀 Installation

### For cPanel Hosting

1. **Extract the ZIP file**
   - Download and extract `jackson-wholesale-complete.zip`

2. **Upload to cPanel**
   - Log in to your cPanel
   - Open File Manager
   - Navigate to `public_html`
   - Upload ALL files from this package
   - Make sure to upload the images folder

3. **Set Permissions**
   - Right-click `index.html`
   - Select "Change Permissions"
   - Set to `644`
   - Right-click images folder
   - Set to `755`

4. **Access Your Site**
   - Visit your domain
   - Everything should work perfectly!

### For Vercel

1. **Create GitHub Repository**
   - Go to [github.com](https://github.com)
   - Create new repository
   - Upload all files

2. **Deploy to Vercel**
   - Go to [vercel.com](https://vercel.com)
   - Sign up with GitHub
   - Import your repository
   - Click "Deploy"
   - Your site is live!

### For Other Hosting

1. Extract the ZIP file
2. Upload all files to your hosting's public directory
3. Visit your domain

## 📞 Support

### If Images Don't Show
- Check file permissions (755 for folders, 644 for files)
- Verify all image files are uploaded
- Clear browser cache
- Check browser console for errors

### If Form Doesn't Work
- Ensure WhatsApp is installed on your device
- Check your phone number is correct
- Test on a different browser

### If Site Looks Wrong
- Clear browser cache (Ctrl+Shift+Delete)
- Try a different browser
- Check file permissions
