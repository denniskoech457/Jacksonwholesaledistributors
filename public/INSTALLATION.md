## 🚀 Installation Methods

### METHOD 1: cPanel (Recommended for Most Users)

#### Step 1: Extract the ZIP File
- Download `jackson-wholesale-complete.zip`
- Right-click and select "Extract All"
- Or use 7-Zip, WinRAR, or your system's unzip tool

#### Step 2: Log in to cPanel
- Go to your hosting provider's website
- Log in to cPanel with your credentials
- Look for "File Manager" icon

#### Step 3: Navigate to public_html
- Click "File Manager"
- You should see a folder called `public_html`
- Double-click to open it
- This is where your website files go

#### Step 4: Upload All Files
- Click "Upload" button
- Select all files from the extracted folder:
  - index.html
  - All PNG files (images)
  - All JPG files (photos)
- Wait for upload to complete (may take 2-5 minutes)

#### Step 5: Set Permissions
- Right-click on `index.html`
- Select "Change Permissions"
- Set to `644`
- Right-click on each image file
- Set to `644`

#### Step 6: Test Your Site
- Open your web browser
- Go to your domain (e.g., `yoursite.com`)
- Your website should load!

### METHOD 2: Vercel (Free, Fast, Easiest)

#### Step 1: Create GitHub Account
- Go to [github.com](https://github.com)
- Click "Sign up"
- Create free account

#### Step 2: Create Repository
- Click "+" menu → "New repository"
- Name: `jackson-wholesale`
- Description: "Jackson Wholesale Distributors Ltd"
- Choose "Public"
- Click "Create repository"

#### Step 3: Upload Files to GitHub
- Click "Add file" → "Upload files"
- Drag and drop all files from the extracted folder
- Or click to select files
- Click "Commit changes"

#### Step 4: Deploy to Vercel
- Go to [vercel.com](https://vercel.com)
- Click "Sign up"
- Choose "Sign up with GitHub"
- Authorize Vercel
- Click "New Project"
- Select `jackson-wholesale` repository
- Click "Import"
- Click "Deploy"

#### Step 5: Your Site is Live!
- Vercel will show you your live URL
- Usually: `jackson-wholesale.vercel.app`
- Share this link with customers!

#### Step 6: Add Custom Domain (Optional)
- In Vercel, go to "Settings" → "Domains"
- Add your domain (e.g., `jacksonwholesale.ug`)
- Follow DNS instructions
- Your site will be at your custom domain

## 🔧 Troubleshooting

### Images Don't Show

**Problem**: Images appear as broken icons

**Solutions**:
1. Check file permissions (should be 644)
2. Verify all image files were uploaded
3. Check file names match exactly
4. Clear browser cache (Ctrl+Shift+Delete)
5. Try different browser

### Website Shows 404 Error

**Problem**: "Page not found" error

**Solutions**:
1. Verify `index.html` is in `public_html`
2. Check file permissions (644)
3. Try accessing `yoursite.com/index.html`
4. Contact hosting support

### Form Doesn't Submit

**Problem**: Clicking submit doesn't work

**Solutions**:
1. Ensure WhatsApp is installed on your device
2. Check phone number is correct (0768 980 819)
3. Try on different device
4. Check browser console for errors

### Site Looks Broken

**Problem**: Styling looks wrong, layout is messed up

**Solutions**:
1. Clear browser cache (Ctrl+Shift+Delete)
2. Try different browser
3. Check file permissions
4. Verify CSS is loading (check browser console)

### Slow Loading

**Problem**: Website takes too long to load

**Solutions**:
1. Images are large (62 MB total) - this is normal
2. Use Vercel for faster global CDN
3. Compress images if needed
4. Check internet connection
